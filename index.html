Arduino Connection

First Make sure to load the Arduino code on your Arduino: LIDAR-Lite Brick Breaker on Github

Choose Serial Port


If you don't see the serial port for your Arduino, try unplugging and replugging the USB and hitting "Reload Connections". This can also help if the serial port has issues during gameplay (you'll see a "reload app" button on the game screen).

LIDAR-Lite Calibration

Be sure to mount the LIDAR-Lite on the left side of your computer.

Before we start the game we need to set a leftmost and rightmost distance our hand will be in relation to the sensor, this will help us calibrate the swing for the paddle. Ideally the leftmost distance will be 10cm or greater.
